PUI: Podium UI Framework

Read this documentation and implement your UI.

Global

원하는 코드를 pui 클래스로 감싸면 됩니다.

<div class="pui">
    여기에 코드를 넣으세요
</div>

Text

글을 text 클래스로 감싸면 문단 간격이 제목 및 문서에 맞게 조절됩니다.

Box

Podium에서 공통적으로 쓰이는 박스를 쓰고 싶을 때는 box 클래스를 사용하세요.

Code Example

<div class="box">
    <div class="text">
        Hello, world!
    </div>
</div>

Result

Hello, world!
